Lesson 9: Array List

When working with arrays, most often knowing the length of the array is important. Array has a length property that returns the length of the array.
To get this property, use the syntax: arrrayname.length

For example:
"String[] stringlist = {"Hello","world"}"
"int x = stringlist.length;"
The variable x will hold the length of the array. In this example, x will be 2.

Another important thing when working with array is accessing one element in the array. To get an elelment in an array, use the syntax: arrayname[index]
The index is the position the element is in in the array.

For the eample above, the string "Hello" will be at index 0.

The same syntax is use to resign an index to a new element.

For example: "stringlist[1] = "today";"
This will change the stringlist array to be {"Hello", "today"}
