Lesson 9: Array List

What is an array? An array is a container object that holds a fixed amount of elements that are the same data type.

There are three ways to create an array.

  • To create an emtpy array use the syntax: data-tyhpe[] arrayname;

    For example: "String[] stringlist;" This create a emtpy array that can only hold String type data.

  • To create an array with length, use the syntax: data-tyhpe[] arrayname = new data-type[length];

    For example: "String[] stringlist = new String[10];" This create a array that has 10 indexs and can only hold string data types. Remeber that array indexs start at 0.

  • To create an array with elemts in it already, use the syntax: data-tyhpe[] arrayname = {element0, element1, ..};

    For example: "String[] stringlist = {"Hello","world"}" This create a array with a length of 1 that has the strings "Hello" and "world" in it. Remeber that array indexs start at 0.
