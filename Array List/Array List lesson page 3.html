Lesson 9: Array List

Multi-dimensional array are an arrays with array elements inside of it. For example: {{"1","2","3"},{"a","b","c"}}

Creating multi-dimensional array are similar to creating a normal array. The only difference is the number of square brackets[ ] you use.
- Normal array use only one sqaure bracket: String[ ] stringlist
- Two- dimensional array use two sqaure brackets: String[ ][ ] stringmatrix

The same is for accessing elements in a multi-dimensional array.
- Normal array use only one sqaure bracket: stringlist[index]
- Two- dimensional array use two sqaure brackets: stringmatrix[index1][index2]

As for the length property for multi-dimensional array, be careful becasue has different lengths. There is a length for the main array that holds the other array elements. Then there is the length for each array elements.

For example: int[ ][ ] nummatrix = {{1, 2, 3}, {4, 5, 6}};
int x = nummatrix.length;
int y = numatrix[0].length;
x will be 2 becasue there is two array elements. y will be 3 becasue there are 3 numbers in each array element.
