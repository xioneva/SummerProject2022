Lesson 6: Comparison

In this lesson, you'll learn about how to use Java Operator to compare and check on the variables and values which return True or False:

For both Integer and variables, we often use the operator:
     > : greater than
     < : less than
     == : equal
     >= : greater or equal
     <= : less or equal
Examples:
     if (1 < 2) -> will result True
     if (5 > 5) -> will result False
     if (7 >= 7) -> will result True
