Lesson 6: Comparison

There are also some Logical Opperators that are often used in comparison and checking:

     && : And Operator -> Yield True if both copmarisions are True
     || : Or Operator -> Yield True if one of the copmarisions is True

Examples:
     if (1 < 2 && 5 > 4) -> True (because both comparison are True)
     if (3 > 2 && 3 <= 1) -> False (because one comparison yield False)

     if (3 > 2 || 3 <= 1) -> True (because one of the comparision is True)
