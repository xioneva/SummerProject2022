Lesson 7: Condition If/Else

In this lesson, you'll learn about the Condition and Execution of the Program
The condition If/Else often follow the General rule:

If (condition) {
     This part of program will run when if condition statement is true
} Else {
     This part of the program will run when if condition statement is false
}

Example:
a = 5;
If (4 < a) {         //check condition to see if the statement is true
     System.out.println("Hello");         //Execute when condition is true
}
Else {
     System.out.println("World");         //Execute when condition is false
}
