Lesson 7: Condition If/Else

There is also another condition call Else if which do another checking when the if condition statement is False.

They follow the General rule:

If (first condition) {
     This part of program will run when the first condition statement is true
} Else if (second condition) {
     This part of program will run when if the first condition statement is false but the second
     condition is true
} Else {
     This part of the program will run when if all condition statement is false
}
