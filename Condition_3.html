Lesson 7: Condition If/Else

Example:
a = 5;
If (4 < a) {         //check condition to see if the statement is true
     System.out.println("Hello");         //Execute when first condition is true
}
Else if (4 == a) {         //check when first condition is false
     System.out.println("My");         //Execute when second condition is true
} Else {
     System.out.println("World");         //Execute when all conditions are false
}
