Lesson 12: File I/O

As you can saw, there is not method in the File class to read the content of the file. So how would you read the content of a file?

You will have to use a scanner. To use a scanner you will need to import the Scanner class. After importing the scanner class, you will also need to create a scanner object.
For example: Import java.until.Scanner;
Scanner [name] = new Scanner([File oject name]);
