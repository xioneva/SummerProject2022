Lesson 12: File I/O

In this lesson, you'll learn about working with files in Java. For Java file handing, you will need to import the java.io package: import java.io.File;

To use the file class, you will need to create an object of the file class with the filename or directory name.
For example: File [name] = new File("filename.txt");
