Lesson 10: Iteration over a List

In this lesson, you'll learn about What is Iteration and How do we Iterate over a list.

What is Iteration?
        In Java, iteration is a technique used to sequence through a block of code repeatedly until a specific condition either exists or no longer exists. Iterations are a very common approach used with loops.

How to Iterate over a list?
        We often use For loop in order to Iterate over a list. First, we intialize and variable index contain value 0. Then we give the condition where index must be less than the length of the list -1.

Example:
list = [0,0,0,0,0];
For (int index = 0; index < list.length; index++) {
     list[index] = index;
}
this will result list = [0, 1, 2, 3, 4]
