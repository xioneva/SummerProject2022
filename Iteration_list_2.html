Lesson 10: Iteration over a List

For nested List, We often use double For loop in order to interate entire nested list:

Example:
list = [[0, 0, 0], [0, 0, 0], [0, 0, 0]]

For (int i = 0; i < list.length; i++) {
     For (int j = 0; j < list.length; j++) {
         list [i] [j] = i + j;
}
this will result list = [[0, 1, 2]; [1, 2, 3]; [2, 3, 4]]
