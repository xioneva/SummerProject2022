Lesson 3: Arithmetic Expressions

  • Addition
    • - The '+' operator can add any two numeric values together. It can also add variables together as long as those variables store a numerical value.
    - Examples:
    int num = 1 + 2; //num is 3
    int number = 1;
    int val = number + 2; //val is 3
    int valueOne = 1;
    int valueTwo = 2;
    int sum = valueOne + valueTwo; //sum is 3
