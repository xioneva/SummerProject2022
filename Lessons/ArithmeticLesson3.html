Lesson 3: Arithmetic Expressions

  • Multiplication
    • - The '*' operator multiplies any two numerical values together. It can also multiply variables as long as they store numerical values.
    - Examples:
    int num = 2 * 1; //num is 2
    int number - 2;
    int val = number * 1; //val is 2
    int valueOne = 2;
    int valueTwo = 1;
    int product = valueOne * valueTwo; //product is 2
