Lesson 3: Arithmetic Expressions

  • Division
    • - The '/' operator divides any two integer values together, but this is not a typical division for integers. Because ints cannot hold decimal values, dividing two integers gives the quotient but without a decimal. A way around this is to store the values of the integers in a float/double, or cast them as a float/double. Casting is simply putting the desired type in parentheses before the value.
    - Examples:
    int num1 = 1 / 2; //num1 = 0
    int num2 = 2 / 2; //num2 = 1
    int num3 = 10 / 6; //num3 = 1
    double num4 = 1.0 / 2.0; //num4 = 0.5
    double cast = (double) 1 / 2; //cast = 0.5
