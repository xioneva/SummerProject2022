Lesson 3: Arithmetic Expressions

  • Modulo
    • - The '%' operator is used to give the modulus of two values. This can be useful in some situations where you would need to know the remainder of two values.
    - Examples:
    int num = 10 % 6; //num = 4
    int value = 10 % 5; //value = 0
