Lesson 2: Floating Point

  • Variable Types
    • - Floating point numbers (numbers with a decimal in them) are represented using either float or double as a variable type. They can only hold floating point values, so just setting an integer value would not work. If you want some integer value to be represented as a floating point value, simply put a .0 after the integer.
    - Examples:
    double number = 123.456;
    float value = 123.456;
    double num = 1.0;
