Lesson 1: Variables

  • Defining the type (naming comes later)
    • - There are many types of variables in Java. Here are some of the most common types:
    - int: integers
    - char: a character (valued using ' ')
    - boolean: a boolean value of either true or false
    - String variables and floating point numbers come later
    - Examples:
    int value;
    int number = 1;
    char letter = 'a';
    boolean b = true;
