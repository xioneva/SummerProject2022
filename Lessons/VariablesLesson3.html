Lesson 1: Variables

  • Assigning Variables Values
    • - Variables can be assigned values as they are defined, or after they are defined. Variables can be assigned after any number of lines of code if they are assigned after it is defined. Code is read from top to bottom, so whichever value comes last will be the one stored in the variable. Variables cannot store multiple values. They should only be assigned values related to the type they are (ints can only be assigned integers, etc.).
    - Examples:
    int number = 1; //assigned as it is defined
    int integer;
    integer = 1; //assigned after it is defined
    char character;
    //Some more lines of code
    character = 'a';
    character = 'b'; //b overwrote a as the value of character
