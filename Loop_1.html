Lesson 8: Loop(For/While)

In this lesson, you'll learn about the two commmon iterations in Java: For and While loop

In For loop, we often follow the general rules:

For (initalize variable value, condition, accumulator) {
     This part of program will run until the accumulator cause the condition to be False
}

Example:
For (int i = 0; i < 10; i++) {
     System.out.println("Hello World");
}

This program will print Hello World 10 times because the For loop use the intialize variable value i and compare it with the condition. When the condition is True, the program will continue execute until the condition is False
