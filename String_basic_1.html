Lesson 4: String Basic

In this lesson, we will learn about the string in Java.

What is String?
     String is a variable type, used to store text. The String
     variable is initialized with double quotes around characters.

Example:
     String a = "Hello World";
     System.out.println(a);
-> Hello World

In this example, the String variable "a" store the the any text that we intialize. When we call the variable, it print out the text that we stored intially
