Lesson 4: String Basic

There are some other method to access the String in the variable:

- length(): help to get the length of the text inside the string.

- indexOf(): help to get specific index contain in the string given character.

- charAt(): help to get the specific character in the string given the index.

- toLowerCase(): help to convert the string to lower case.

- toUpperCase(): help to convert a string to upper case.

- concat(): help to append 2 String together

- split() : help to split a string into multiple strings and returns and array of string

- join(): simliar to concat() method, help to join multiples strings into one string. But it may cause overloaded.

- isEmpty(): help to check if the String is Empty or not by returning a boolean
